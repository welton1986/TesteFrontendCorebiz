## ✔ Como baixar o projeto


```bash

# Clonar repositório
$ git clone https://github.com/welton1986

# Entrar no diretoório
$ cd TesteFrontEndCorebiz

#Instalar as dependências
$ yarn install

#Instalar o projeto
$ yarn start

```
